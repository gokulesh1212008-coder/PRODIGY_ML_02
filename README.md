# 🛒 Customer Segmentation using K-Means Clustering

## 📌 About the Project

Customer segmentation is one of the most important applications of Machine Learning in business analytics. In this project, I implemented the **K-Means Clustering** algorithm to group customers based on their purchasing behavior.

The model segments customers using:

- 💰 Annual Income (`Annual Income (k$)`)
- 🎯 Spending Score (`Spending Score (1-100)`)

These customer segments can help businesses understand customer behavior and design targeted marketing strategies.
